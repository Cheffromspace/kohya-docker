# Docker image for Kohya_ss

## Installs

* Ubuntu 22.04 LTS
* CUDA 11.8
* Python 3.10.6
* Torch 2.0.1
* xformers 0.0.20
* [Kohya_ss](https://github.com/bmaltais/kohya_ss) v21.7.10
